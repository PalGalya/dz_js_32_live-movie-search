# Застосунок пошуку фільмів

Це веб-застосунок для пошуку фільмів з використанням OMDb API. Застосунок дозволяє користувачам шукати фільми за назвою та відображає результати у вигляді сітки постерів.

## Особливості

- Пошук фільмів за назвою
- Відображення постерів фільмів
- Дебаунс запитів для оптимізації роботи з API
- Можливість додавати нові фільми до існуючого списку

## Структура проєкту

```
index.html          - HTML-структура застосунку
index.js            - Головний JavaScript файл
styles.css          - CSS-стилі для застосунку
vitest.config.js    - Конфігурація Vitest для тестування
vitest.setup.js     - Налаштування Vitest
assets/
  img/
    no-image.png    - Зображення-заповнювач за відсутності постера
src/                - Директорія для вихідних файлів
```

## Технології

- Vanilla JavaScript
- HTML5
- CSS3
- OMDb API для отримання даних про фільми
- Vitest для тестування

## Як використовувати

1. Відкрийте `index.html` у веб-браузері
2. Введіть назву фільму в поле пошуку (мінімум 4 символи)
3. Зачекайте результати пошуку (застосовується затримка 2 секунди)
4. Для додавання нових фільмів до існуючих результатів використовуйте чекбокс "Додавати фільми до вже існуючого списку"

## API

Застосунок використовує [OMDb API](http://www.omdbapi.com/) для пошуку фільмів. API-ключ вже вбудований в застосунок.

## Розробка

Для запуску тестів використовуйте:

```bash
# Запуск тестів один раз
npm test

# Запуск тестів у режимі відстеження змін
npm run test:watch
```

## Примітки

- Пошук починається тільки після введення не менше 4 символів
- Для оптимізації запитів до API використовується дебаунс із затримкою 2 секунди
- Якщо постер недоступний, буде відображатися заповнювач "no-image.png"
